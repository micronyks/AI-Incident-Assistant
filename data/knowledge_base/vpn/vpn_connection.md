# VPN Connection Issue

## Symptoms

- Unable to connect to VPN
- Authentication failed
- VPN disconnects frequently
- VPN stuck on connecting

## Root Causes

- Expired password
- Cisco Secure Client outdated
- Firewall blocking VPN ports
- MFA not configured
- Internet connectivity issue

## Resolution

1. Verify internet connection.
2. Reset corporate password.
3. Update Cisco Secure Client.
4. Restart VPN service.
5. Flush DNS.
6. Reboot machine.

## Escalation

Assign to Network Operations Team if issue persists after 30 minutes.
